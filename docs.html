czRadio jQuery plugin

Introduction

czRadio is a jQuery plugin that allows you to turn default browser radio into much more attractive.

The plugin is usable not only for end users, but also for developers. czRadio has a lot of configuration options so you can modify its behaviour and appearance. From this version the core CSS file is separated from the CSS that provides radio's appearance, and now you are able to easily create new skins for the plugin.

Examples

Please view demo page to see the possibilities of czRadio.

Installation

Please follow these instructions to install czRadio:

  • Download and unpack the archive.
  • Include jQuery and plugin files to your web page: 
    <script type="text/javascript" src="/js/jquery-1.4.2.min.js"></script>
    <script type="text/javascript" src="/js/jquery.czRadio.js"></script>
  • Include core or your skin CSS files to your page: 
    <link rel="stylesheet" type="text/css" href="images/style.css" />
    <link rel="stylesheet" type="text/css" href="images/skin_name.css" />
  • Done! Now make your radios look and behave sexy! 
    $("#radiolist").czRadio();
$("#list").czRadio({
    changeCallback: function(){
        alert('changed')
    }
});

Usage and configuration options

czRadio has a number of configuration options that are passed to the plugin in the form of JavaScript object.

The full list of options is:

className String, Default: null
The class name of radio list wrap.
inputName String, Default: null
The name of radio you need.
initCallback Funtion, Default: null
function that is called at the end of constrictor.
forbidCallback Funtion, Default: null
function that is called when the disabled radio is selected.
changeCallback Funtion, Default: null
function that is called when the selection of radio is changed.

In this version I have separated core CSS and presentational CSS, so now it's possible to create new skins for czRadio. The download package contains one example skin. Feel free to create your own based on it.

Browser compatibility

czRadio has been tested and works on the following browsers:

  • Internet Explorer 6.0 +
  • Chrome
  • Firefox
  • Opera
